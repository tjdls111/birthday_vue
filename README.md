# birthday_vue
vue 공부할 겸 친구 생일 축하할 겸 ~~만든~~ 만들고 있는 사이트!

- 배포 링크: https://mgbirthday.netlify.app/ 
- 기능: 메인 화면, 선물 추천하는 화면, 축하 영상으로 보내는 화면
- 사용 도구 : vue, bootstrap
- 친구들 생일마다 업그레이드할 예정...!!
